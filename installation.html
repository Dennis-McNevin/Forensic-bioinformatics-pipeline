<h2>Installation</h2>
<p> To install the MPS Forensics application:
<ol>
<li>
Download https://www.dropbox.com/s/nhi8e9vzfgwga0q/install_mpsforensics.sh
</li>
<li>
The application requires at least 20 GB of disk space to accomodate multiple indexes of the human genomic reference.
</li>
<li>
Run this script as administrator/with administrator privileges. <br/>
It will take a long time as it has to download the lobSTR genomic index as well as the hg19 complete human reference genome as indexed with BWA. (If you already have these downloaded you can comment out the appropriate ‘wget’ lines with a # symbol, but will need to create links to your own copies which is a non-trivial task).
</li>
<li>
Once this is complete, the administrator will need to add the ngsforensics group to each user account that should be allowed access to the application. This can be done from the command line with:
# usermod -a -G ngsforensics username
Or through the graphical interface:
On Ubuntu:
On Fedora:
</li>
<li>
Lastly the administrator should create a link for each user of the application to /home/ngsforensics/forensicsapp/start.sh which is the launcher for the application. This launcher makes sure that the database and web service are running (to support the viewer application) and after a 30 second delay starts the main application and MPS pipeline launcher.
</li>
</ol>
<br/>
</p>

<h2>Using the MPS Forensics application</h2>
<p><br/>Running the MPS Forensics application is as simple as double-clicking the link provided by your administrator. Ideally this is placed on the desktop, or somewhere else that is equally obvious to find.</p>
<p>When you start the application it will first ensure that the required database and web server are running. These are needed by the visualisation component of the pipeline. After a 30 second delay (to guarantee enough time for these services to start cleanly) the main application is started.</p>
<p>The application is composed of a number of tabs. The first of these is a welcome page that also contains three buttons, which launch the viewer service in the Firefox web browser.</p>

Here is the link to our installer for the MPS Forensics Pipeline system we have been developing: https://www.dropbox.com/s/wpvz8thrao0q8sw/install_mpsforensics.sh To install it on your own Ubuntu 14.04.4 LTS or Ubuntu 16.04 LTS installation: 1) You can either download it directly on the Ubuntu computer using a web browser, or copy it using a memory stick from another machine. 2) Place the install_mpsforensics.sh file on the Desktop. You will need to use the command line to install it, but this isn't too hard to do... 3) On the left hand side of the screen is the Task Bar. At the top of the Task Bar is a purple and white icon, which when you move mouse over it, says "Search your computer". Click on this icon. 4) A window will appear next to the button. At the top of this is an empty text box. Click in this, and type (without the quotes) "terminal". 5) Now a header will appear directly under the text box that reads "Applications", and under this is a black and white icon and the words underneath read "Terminal". Click on this icon. 6) A command-line terminal should appear and present you with a prompt that looks like: username@computername"~$ The ~ symbol means you are in your home folder. 7) At the $ prompt type the following and press the Enter key (don't type the $ sign): $ bash Desktop/install_mpsforensics.sh 8) 6 lines of text will appear, the last one reads: [sudo] password for username: The username shown will be your username, not the actual word. Here you need to type the password for your account and press the Enter key when you are done. This lets the installer set up system-wide software libraries. When you type the password the characters will not appear on the screen. This is to protect you from people watching over your shoulder, but it also means it's easy to mistype it. Don't worry if you get it wrong, just try again. If you get it wrong three times the installation will fail, but you can simply repeat instruction (7) to try again. 9) The installation should run to completion. Depending on the speed of your internet connection the installation should take between 5 and 20 minutes. If the a download breaks then just repeat from step (7) to try again. The installer only downloads what it doesn't already have. If it completes successfully you see one the last messages "Installation of MPS Forensics pipeline complete". 10) There will now be two new links on your Desktop. a) The first is an icon with the name "MPSforensics.sh" - double clicking this icon will start the MPS Forensics software, as well as the Meteor web engine and the IGV genome browser. b) The second is link with the name "MPS_results" - double clicking this will open the /home/username/mpsforensics/results folder, which is where all the pipeline runs are placed by default.
