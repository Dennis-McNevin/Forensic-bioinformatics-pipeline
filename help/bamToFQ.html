Pipeline synopsis

All pipelines available in this system are capable of taking FASTQ, FASTQ.GZ or BAM files as input, but if the user desires, they can use this simple pipeline to convert BAM files to FASTQ.GZ format for their convenience.

The pipeline relies on Picard Tools, which is also required if GATK is ever implemented as an alternative SNP caller.
